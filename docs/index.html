Entenda o jogo

Pac man é um jogo do inicio da decada de 80, que se popalarizou por todo o mundo se tornando um dos mais adorados jogos de fliperama do mundo.
Terão 4 fantasmas que fazem você perder. Cada fantasma tem sua função. O vermelho, por exemplo, irá tentar te seguir durante toda a partida, até conseguir te pegar. Seu objetivo é comer todas as pastilhas pelo mapa até acabar acumulando o máximo de pontos. O jogo só encerra quando você perder todas suas vidas, mostrando sua pontuação e te dando a opção de salva-lá em seu historico.

O item "come fantasma"⚪ lhe dará 10 segundos para comer o maximo de fantasmas possiveis te deixando invulnerável e, dando 200, 400, 800 e 1600 pontos, respectivamente, para cada fantasma comido durante esse periodo.

Instruções:

Use as setas para mover seu personagem.
Os fantasmas tentarão te seguir tome cuidado!
Coma todas as pastilhas para ganhar pontos, continue comendo até ser pego.

Como jogar

Instalando o python :

Primeiramente, acesse o site oficial do Python aqui e faça o download do instalador de acordo com o seu sistema operacional (Windows, Mac ou Linux).

Após baixar o instalador, execute-o e siga as instruções do assistente de instalação. É importante escolher a opção de "Adicionar Python ao PATH" durante a instalação, para que o Python possa ser acessado facilmente a partir do terminal (linha de comando).

Depois de concluir a instalação, abra o terminal (no Windows, pressione a tecla Windows + R e digite "cmd" na caixa de diálogo; no Mac, abra o Terminal no Launchpad ou na pasta Aplicativos > Utilitários).

Digite "python" no terminal e pressione Enter. Se tudo estiver correto, você deverá ver a versão do Python instalada e o prompt de comando do Python (>>>).

Parabéns, você instalou com sucesso o Python no seu computador! Agora você tem tudo quase tudo que precisa para jogar o jogo, só falta o baixar o zip do jogo.


Baixar zip do jogo :

Clique aqui para ir no repositório git do projeto, siga o readme para baixar e se divirta jogando. Ou se já tiver lido o tutorial, e souber o que fazer aperte aqui para baixar o zip do jogo
